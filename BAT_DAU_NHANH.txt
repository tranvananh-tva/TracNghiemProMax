✨ BƯỚC 3: TẠO QUIZ ĐẦU TIÊN
═══════════════════════════════════════════════════════════════

   1. Click "Tạo đề thi" (nút tím ở header)
   
   2. Nhập thông tin:
      ┌─────────────────────────────────────────────────┐
      │ Tên quiz: Kiểm tra Toán học                     │
      │ Mô tả: Bài kiểm tra cơ bản                      │
      └─────────────────────────────────────────────────┘
   
   3. Nhập câu hỏi:
      ┌─────────────────────────────────────────────────┐
      │ Câu 1: 2 + 2 = ?                                │
      │ A. 3                                             │
      │ B. 4                                             │
      │ C. 5                                             │
      │ D. 6                                             │
      │                                                  │
      │ Câu 2: 5 x 3 = ?                                │
      │ A. 8                                             │
      │ B. 15                                            │
      │ C. 20                                            │
      │ D. 25                                            │
      └─────────────────────────────────────────────────┘
   
   4. Nhập đáp án:
      ┌─────────────────────────────────────────────────┐
      │ Câu 1: B                                         │
      │ Câu 2: B                                         │
      └─────────────────────────────────────────────────┘
   
   5. Click "Xử lý & Tạo quiz"
   
   6. ✅ Xong! Quiz đã được tạo

🎯 BƯỚC 4: LÀM BÀI
═══════════════════════════════════════════════════════════════

   Cách 1: Click vào quiz card ở trang chủ
   
   Cách 2: 
   1. Vào tab "Làm Bài" (sidebar)
   2. Chọn quiz từ dropdown
   3. Click "Bắt Đầu Làm Bài"
   
   Giao diện làm bài:
   ┌─────────────────────────────────────────────────────────┐
   │  SIDEBAR TRÁI     │    MAIN CONTENT    │  SIDEBAR PHẢI  │
   │  ├─ Thông tin     │    ├─ Câu hỏi     │  ├─ Grid câu   │
   │  ├─ Timer         │    ├─ Đáp án      │  │  1 2 3 4 5  │
   │  ├─ Settings      │    └─ Nút điều    │  │  6 7 8 9 10 │
   │  └─ Progress      │       hướng       │  └─ Click số    │
   └─────────────────────────────────────────────────────────┘

═══════════════════════════════════════════════════════════════

🏆 BƯỚC 5: XEM KẾT QUẢ
═══════════════════════════════════════════════════════════════

   1. Sau khi nộp bài, tự động chuyển sang tab "Kết Quả"
   
   2. Xem:
      ✅ Điểm số (X/10)
      ✅ Số câu đúng/sai
      ✅ Thời gian làm bài
      ✅ Tỷ lệ đúng (%)
   
   3. Click "Xem Lại Bài Làm" để review từng câu
