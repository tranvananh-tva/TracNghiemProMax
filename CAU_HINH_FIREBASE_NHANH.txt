⚠️ LƯU Ý: API key bạn cung cấp không đúng định dạng Firebase.
Để người dùng từ các thiết bị khác xem được bài chia sẻ, bạn cần:

═══════════════════════════════════════════════════════════════════
📋 BƯỚC 1: TẠO FIREBASE PROJECT (5 PHÚT)
═══════════════════════════════════════════════════════════════════

1. Truy cập: https://console.firebase.google.com/
2. Đăng nhập bằng Gmail
3. Click "Add project" → Đặt tên "TracNghiemPro"
4. Tắt Google Analytics → Click "Create project"

═══════════════════════════════════════════════════════════════════
📋 BƯỚC 2: TẠO FIRESTORE DATABASE (2 PHÚT)
═══════════════════════════════════════════════════════════════════

1. Click "Firestore Database" ở menu trái
2. Click "Create database"
3. Chọn "Start in test mode" → Next
4. Chọn location: "asia-southeast1" → Enable

═══════════════════════════════════════════════════════════════════
📋 BƯỚC 3: CẤU HÌNH RULES (1 PHÚT)
═══════════════════════════════════════════════════════════════════

1. Click tab "Rules"
2. Thay thế bằng code sau:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /shared-quizzes/{document=**} {
      allow read, write: if true;
    }
  }
}

3. Click "Publish"

═══════════════════════════════════════════════════════════════════
📋 BƯỚC 4: LẤY CONFIG (2 PHÚT)
═══════════════════════════════════════════════════════════════════

1. Vào Project Settings (biểu tượng bánh răng)
2. Scroll xuống "Your apps"
3. Click biểu tượng Web (</>)
4. Đặt tên: "TracNghiemProWeb"
5. Click "Register app"
6. Copy đoạn firebaseConfig

VÍ DỤ CONFIG:
const firebaseConfig = {
  apiKey: "AIzaSyC1234567890abcdefghijklmnop",
  authDomain: "tracnghiempro-12345.firebaseapp.com",
  projectId: "tracnghiempro-12345",
  storageBucket: "tracnghiempro-12345.appspot.com",
  messagingSenderId: "123456789012",
  appId: "1:123456789012:web:abcdef1234567890"
};

═══════════════════════════════════════════════════════════════════
❓ KHẮC PHỤC SỰ CỐ
═══════════════════════════════════════════════════════════════════

Lỗi: "Firebase initialization failed"
→ Kiểm tra config đã copy đúng chưa
→ Kiểm tra Firestore đã enable chưa

Lỗi: "Permission denied"
→ Kiểm tra Rules đã publish chưa
→ Rules phải có: allow read, write: if true;

Lỗi: "Missing or insufficient permissions"
→ Vào Firestore → Rules → Publish lại
