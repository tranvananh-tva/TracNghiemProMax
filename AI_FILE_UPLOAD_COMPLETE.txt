📁 HỖ TRỢ FILE
═══════════════════════════════════════════════════════════════

1. FILE TXT (.txt)
   ✅ Hỗ trợ: 100%
   ✅ Đọc: Hoàn hảo
   ✅ Encoding: UTF-8
   ✅ Tốc độ: Nhanh
   
   Cách dùng:
   - Lưu nội dung bài học vào file .txt
   - Upload file
   - AI tự động đọc và tạo quiz

2. FILE WORD (.docx, .doc)
   ⚠️ Hỗ trợ: Cơ bản (70%)
   ⚠️ Đọc: Text thuần
   ⚠️ Hạn chế: Không đọc hình ảnh, bảng phức tạp
   ⚠️ Khuyến nghị: Copy nội dung và dán trực tiếp
   
   Cách dùng:
   - Upload file .docx
   - Hệ thống cố gắng extract text
   - Nếu không đọc được → Copy thủ công
   
   Lưu ý:
   - File Word phức tạp có thể không đọc được
   - Nên dùng file Word đơn giản (chỉ text)
   - Hoặc copy nội dung và dán trực tiếp

3. FILE PDF (.pdf)
   ❌ Hỗ trợ: Chưa đầy đủ
   ❌ Đọc: Yêu cầu thư viện bổ sung
   ❌ Khuyến nghị: Copy nội dung từ PDF và dán
   
   Cách dùng:
   - Mở file PDF
   - Copy nội dung (Ctrl + A, Ctrl + C)
   - Dán vào textarea
   - Click "Tạo Quiz Bằng AI"
   
   Lý do:
   - Đọc PDF cần thư viện pdf.js (nặng)
   - Để giữ ứng dụng nhẹ, chưa tích hợp
   - Sẽ bổ sung trong phiên bản sau

⚙️ THUẬT TOÁN ĐỌC FILE
═══════════════════════════════════════════════════════════════

1. VALIDATION:
   ├─ Kiểm tra extension (.txt, .docx, .pdf)
   ├─ Kiểm tra size (max 10MB)
   └─ Thông báo lỗi nếu không hợp lệ

2. ĐỌC FILE TXT:
   ├─ Dùng FileReader API
   ├─ readAsText() với UTF-8
   ├─ Return text content
   └─ Set vào textarea

3. ĐỌC FILE WORD:
   ├─ Đọc file như ArrayBuffer
   ├─ Convert sang Uint8Array
   ├─ Extract text từ binary
   ├─ Clean up và normalize
   └─ Return text (nếu có)

4. ĐỌC FILE PDF:
   ├─ Thông báo cần thư viện
   ├─ Hướng dẫn copy thủ công
   └─ Throw error

5. HIỂN THỊ:
   ├─ Show file info (name, size)
   ├─ Hide upload area
   ├─ Set content vào textarea
   └─ Toast thông báo thành công

═══════════════════════════════════════════════════════════════

💡 TIPS & TRICKS
═══════════════════════════════════════════════════════════════

✅ FILE TXT:
   - Lưu nội dung với encoding UTF-8
   - Tránh ký tự đặc biệt
   - Định dạng rõ ràng, dễ đọc

✅ FILE WORD:
   - Dùng file đơn giản (chỉ text)
   - Tránh bảng, hình ảnh phức tạp
   - Nếu không đọc được → Copy thủ công
   - Format: .docx tốt hơn .doc

✅ FILE PDF:
   - Mở PDF bằng Adobe Reader
   - Select All (Ctrl + A)
   - Copy (Ctrl + C)
   - Paste vào textarea (Ctrl + V)

✅ NỘI DUNG:
   - Ít nhất 200-300 ký tự
   - Câu văn rõ ràng, có ý nghĩa
   - Tránh nội dung quá ngắn
   - Có số liệu, định nghĩa càng tốt

═══════════════════════════════════════════════════════════════

⚠️ HẠN CHẾ & LƯU Ý
═══════════════════════════════════════════════════════════════

⚠️ FILE WORD:
   - Chỉ đọc được text thuần
   - Không đọc được:
     • Hình ảnh
     • Bảng phức tạp
     • Công thức toán học
     • Định dạng đặc biệt
   - Khuyến nghị: Copy thủ công

⚠️ FILE PDF:
   - Chưa hỗ trợ đầy đủ
   - Cần thư viện pdf.js (nặng ~500KB)
   - Khuyến nghị: Copy thủ công
   - Sẽ bổ sung trong tương lai

⚠️ KÍCH THƯỚC:
   - Max 10MB
   - File quá lớn → Lỗi
   - Nên dùng file < 5MB

⚠️ ENCODING:
   - Chỉ hỗ trợ UTF-8
   - File encoding khác có thể lỗi font
   - Nên lưu file với UTF-8
