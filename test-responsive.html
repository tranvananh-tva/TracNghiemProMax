Hướng Dẫn Test

Cách test responsive:

  1. Chrome DevTools: F12 → Toggle Device Toolbar (Ctrl+Shift+M)
  2. Firefox: F12 → Responsive Design Mode (Ctrl+Shift+M)
  3. Manual: Thay đổi kích thước cửa sổ trình duyệt
  4. Real Device: Mở trên điện thoại/tablet thật

Các kích thước cần test:

  • 📱 iPhone SE: 375 x 667
  • 📱 iPhone 12 Pro: 390 x 844
  • 📱 Samsung Galaxy S20: 360 x 800
  • 📱 iPad: 768 x 1024
  • 💻 iPad Pro: 1024 x 1366
  • 💻 Laptop: 1366 x 768
  • 🖥️ Desktop: 1920 x 1080
  • 🖥️ 4K: 2560 x 1440

Checklist:

  • ✅ Layout không bị vỡ
  • ✅ Text dễ đọc
  • ✅ Buttons đủ lớn để click/tap
  • ✅ Không có horizontal scroll
  • ✅ Images scale đúng
  • ✅ Forms dễ sử dụng
  • ✅ Navigation hoạt động tốt
